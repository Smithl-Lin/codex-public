# law2 (AMANI Task Worklog Policy)

## 1. Scope
`law2` applies to every completed task in AMANI Project, including audit, fix, QA, review, and documentation tasks.

## 2. Mandatory Output
After each task is completed, create one worklog markdown file.

## 3. Output Location
All worklog files must be saved in:
- `C:\2025-12-01 Mayo-working\2026-01-22 Business projects\AMANI Project\20260211`

## 4. File Naming
Use timestamped names:
- `WORKLOG_YYYYMMDD_HHMMSS.md`

## 5. Minimum Required Sections
1. Task ID / Task Name
2. Objective
3. Files changed
4. Commands executed
5. Test/validation results
6. Risks / follow-up items
7. Completion time

## 6. Enforcement
A task is not considered complete unless its worklog is present in the required folder.
